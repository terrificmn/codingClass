# reset

업로드 실패할 시  : 업로드를 했는데 아예 반응이 없거나 실패할 경우   

PUSH SW2 버튼을 누르고 있는 상태에서 Reset 버튼을 누르고 뗀다. 마지막으로 PUSH SW2를 뗀다   

그러면 led가 깜빡깜빡 거리는데   
이후 다시 업로드를 진행하면 업로드가 잘 된다
