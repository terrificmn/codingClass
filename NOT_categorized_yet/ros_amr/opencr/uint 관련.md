uint 관련 해당 숫자 (bits를 의미) 해당하는 range가 정해져 있다 

> unsigned integer numbers

| number of bits | min | max        |
| -------------- | --- | ---------- |
| 8              | 0   | 255        |
| 16             | 0   | 65535      |
| 32             | 0   | 4294967295 |
| 64             | 0   | 2^64 - 1   |
