여기에서 CPU라고 나오는 항목이 PIN 번호이다. (라즈베리파이에서는 BCM이라고 나오는 항목)

필요한 핀 넘버를 출력으로 지정해서 input / output 으로 설정해줄 수가 있다
```
gpio -g mode 72 input
```

직접 명령을 입력 ,  핀 넘버 1 을 주면 작동 

```
gpio -g write 72 1
```

핀넘버 0 을 주면 꺼짐
```
gpio -g write 72 0
```
