리눅스 용으로 다운받은 후 압축 풀어서 사용  


postman 말고 insomnia 프로그램도 있지만, 일단 설치는 다운후 dpkg로 해주면 되나, 에러 발생 (트러블 슈팅 필요해서)

## 사용 방법
Workspace를 추가해준다음에   

Method 형식을 선택 (GET, POST 등)..

POST로 테스트 할 경우에는   
Body 선택 후에 raw 선택을 해주고 text 형태에서 JSON 으로 바꿔서 바로 json으로 처리해서 보낼 수가 있다
