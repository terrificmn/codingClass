.box { // selector width: 는 property width: 200px; 의 200px는 values } 로 구성되어 있다 이 중 values 에서 Absolute length units 는 cm(centimeters), mm(millimeters), in(inches), px(pixels) 등 이 중 현실에서는 다양하게 사용하지만 웹에서는 주로 px를 사용 이 중 글자크기는 pt(points)로 사용하지만 이것도 px로 변환되거나 직접 사용됨 px는 웹브라우저의 크기가 변화하더라도 그 크기가 고정적으로 유지된다는 것이고 이로인한 단점은 사용자가 폰트 사이즈를 바꿔도 반응하지 않는 단점이 있다 반면에 Relative length units 는 vw, vh, % 등이 있다 %를 사용하면 컨테츠가 유동적으로 만들 수 있다 주로 사용하는 단위 em: font size of the parent, in the case of typographical properties like font-size, and font size of the element itself, in the case of other properties like width. rem: font size of the root element vw: viewport's width vh: viewport's height %: relative to the parent element
